# Sentiment Analysis using Linear Regression

This project demonstrates sentiment analysis on customer review text
using Linear Regression and TF-IDF feature extraction.

## Features
- Non-structured text processing
- TF-IDF vectorization
- Linear Regression model
- Noisy data handling
- Accuracy evaluation
- Data visualization

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
